02 / C++ Machine Learning NLP

ML Tag Classifier
in C++

For my Programming and Introductory Data Structures course (EECS 280), I built a machine learning algorithm in C++ that predicts the tag attached to a forum post. It uses a bag-of-words technique — tallying word frequencies across posts with known tags using maps, computing a log-probability score per word per tag, then summing those scores across every word in a new post to identify its most likely tag.

87.08% Accuracy

~3,000 Test posts
